# 이번 과제에서 만들어 볼 게임

## 게임 “폴가이즈” 맵 중 하나

언덕 위에서 내려오는 다양한 크기의 슬라임 구체를 피하는 맵이 있습니다. 슬라임 구체를 맞으면 그 즉시 탈락하는 생존 맵입니다.

그 맵을 조금 바꿔서 내려오는 구체를 피하면서 구체 사이에 섞여있는 별을 획득하는 게임입니다. 구체에 부딪치면 원래 탈락하지만 여기서는 체력을 잃습니다. 체력을 잃어도 중간에 나오는 포션을 획득하면 체력을 회복할수 있습니다.

총 레벨은 3단계이고 단계마다 3개의 웨이브가 있습니다.

- 웨이브에 따라 아이템의 갯수와 시간을 조절합니다.
- 레벨에 따라 난이도 상승
    - 예시 (확정 아님)
        - 구체의 스폰 확률을 올린다.
            - 별 아이템의 스폰 확률 감소
        - 장애물을 설치하여 구체의 이동을 방해하거나 구체의 방향을 바꾼다.

## 아이템

- 아이템 공통 기능
    - 스폰후 일정 시간이 지나면 사라진다.
    - 부딪치면 사라진다.
    - 플레이어가 있는 발판에서 떨어지면 사라진다.
- 다양한 크기의 구체 (부딪치면 체력이 깍임)
    - 크기는 다르지만 체력이 닳는 수치는 같다.
        - 체력 : -10
    - 부딪치면 터지면서 사라진다.
- 별 (획득시 점수 +, -)
    - 금 별 : 5 점
    - 은 별 : 3 점
    - 동 별 : 1 점
    - 검은 별 : -1 점
- 체력 회복 아이템
    - 회복 : 30
- 이동속도 감소 포션
    - 이동속도 절반 감소 (중복 획득시 중첩)
    - 중복 획득 시 시간 초기화
- 조작 반전 아이템
    - 조작 반전
    - 중복 획득 시 시간만 초기화
